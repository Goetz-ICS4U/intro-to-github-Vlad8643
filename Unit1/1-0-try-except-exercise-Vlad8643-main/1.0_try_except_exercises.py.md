### Exercise 5
Below is a calculator program that works but has NO error handling. Your job is to add try/except blocks to make it robust and user-friendly.

#### Tasks:
- Fix the menu choice input: What happens if the user types "hello" instead of a number? Add try/except to handle this.
- Fix the number inputs: What happens if the user types "five" instead of 5 when entering numbers? Add try/except to handle this.
- Fix the division: What happens if the user chooses division and enters 0 as the second number? Add try/except to handle this.
- BONUS Challenge: Create a separate function called get_menu_choice() that handles getting a valid menu choice (1-5) with proper exception handling, and returns the choice. Use this function in your calculator.

#### Hints:
- You might need try/except in multiple places!
- Think about WHERE each error could happen
- Make your error messages helpful to the user

```python
def display_menu():
    print("\n=== Calculator Menu ===")
    print("1. Add")
    print("2. Subtract")
    print("3. Multiply")
    print("4. Divide")
    print("5. Exit")

def main():
    """Simple calculator - needs exception handling!"""
    choice = 0
    while choice != 5:
        display_menu()
        choice = int(input("\nEnter your choice (1-5): "))

        # Exits the menu immediately
        if choice == 5:
            print("Goodbye!")
            break 

        # skips to next iteration
        if choice < 1 or choice > 5:
            print("Please enter a number between 1 and 5!")
            continue
        
        num1 = float(input("Enter first number: "))
        num2 = float(input("Enter second number: "))
        
        if choice == 1:
            print(f"Result: {num1 + num2:.2f}")
        elif choice == 2:
            print(f"Result: {num1 - num2:.2f}")
        elif choice == 3:
            print(f"Result: {num1 * num2:.2f}")
        elif choice == 4:
            result = num1 / num2
            print(f"Result: {result:.2f}")
        else:
            print("Please enter a number between 1 and 5!")

# Run calculator
if __name__ == "__main__":
    main()
```
